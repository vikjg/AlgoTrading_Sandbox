# AlgoTrading_Sandbox
Python repo for learning basics of indicators, trading strategies, and backtesting on Bitcoin data. The project is still in it's infancy; i.e. nothing cool beyond the framework has been done yet.
As of now, the mechanisms to pull Kraken data, add indicators, plot candles, set strategies, backtest, and then compare results versus just holding the asset.
The plots can pull in more data and update at a specified time interval too.
